# Apex

A motorcycle riding companion for iOS. Record your rides; get them back as
beautiful keepsakes worth keeping.

Built entirely on Linux — **no Mac in the loop** — using [xtool](https://xtool.sh).
UI is verified by rendering real screenshots on macOS CI and inspecting the
pixels.

## The bar
Transit-level UI/UX. Dark. Kinetic. Premium. If a screen would look out of place
next to the Transit app, it isn't done.

## Features (v1)
- **Ride library** — saved rides as a collection of beautiful objects
- **Ride detail** — one ride, its route on a styled dark map, its trophy stats
- **Live recording** — record a ride, watch the route draw and stats climb
